## Environment

- iOS 17.0+
- macOS 14.0+
- watchOS 10.0+
- Xcode Version 16.0+
- Swift 6.0+

## Setup

```bash
git clone git@github.com:Koshimizu-Takehito/TicTacToe.git
cd TicTacToe
open TicTacToe.xcworkspace
```
